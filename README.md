# 🛍️ E-Commerce Product Listing UI

A fully responsive e-commerce product listing page built using **React.js**.
This project replicates a modern shopping UI with filters, and mobile responsiveness.

---

## 🚀 Features

### 🖥️ Desktop

* Product grid layout
* Sidebar filters (expandable sections)
* Sorting dropdown (Recommended, Price, etc.)
* Clean and modern UI

### 📱 Mobile

* Responsive design
* Slide-in filter panel
* Bottom sort dropdown
* Touch-friendly interactions

---

## 🧩 Components

* **Header** – Navigation, logo, icons
* **Filters** – Accordion filter sections
* **ProductGrid** – Displays all products
* **ProductCard** – Individual product UI
* **Footer** – Footer content

---

## 🛠️ Tech Stack

* React.js
* HTML, CSS (Custom styling)


---
